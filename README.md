# 📰 Fake News Detection System

This project detects whether a news article is Real or Fake using Machine Learning.

## 🚀 Features
- Real-time prediction
- Confidence score
- Clean Streamlit UI

## 🛠️ Tech Stack
- Python
- Scikit-learn
- NLP (TF-IDF)
- Streamlit

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app/app.py
